01

명확한 구조

페이지를 처음 열었을 때 무엇을 보여주고 어디로 이동할지 한눈에 알 수 있습니다.

  • 히어로 섹션에서 메시지와 CTA를 동시에 제시
  • 카드 기반 섹션으로 정보 밀도를 균형 있게 유지
  • 좁은 화면에서는 단일 열로 자연스럽게 재배치
02

기본 접근성

가시적인 포커스와 충분한 터치 영역을 포함해 기본 사용성을 확보했습니다.

  • 버튼과 내비게이션은 44px 이상 클릭 영역 확보
  • 명도 대비가 충분한 배경과 본문 색상 사용
  • 스크린 리더를 위한 내비게이션 라벨 반영
03

즉시 실행

별도 빌드 파이프라인 없이 정적 파일 하나로 빠르게 배포나 검증이 가능합니다.

  • `npm start`만으로 로컬 서버 실행
  • 의존성 없이 저장소 상태를 단순하게 유지
  • 추가 개발 시 정적 페이지를 기반으로 확장 가능
