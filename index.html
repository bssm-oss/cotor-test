Quiet Web Experiment

작고 아름다운 Orbit Garden

아무 웹이나라는 요청에 맞춰, 과장 없이 보기 좋고 모바일에서도 자연스럽게 스크롤되는 단일 페이지를 만들었습니다. 따뜻한 톤, 큰 타이포, 가벼운 유리 질감으로 분위기를 잡았습니다.

분위기 보기 디자인 메모

Morning Light

배경은 단색 대신 겹치는 빛 번짐으로 구성했고, 본문은 충분한 줄간격을 둬서 숨 막히지 않게 읽히도록 처리했습니다.

Soft Motion

상호작용은 버튼의 미세한 상승만 남겨 과하지 않게 정리했습니다. 빠르지만 튀지 않는 움직임입니다.

1

정적 파일 하나로 끝나는 가장 작은 웹 산출물

320px+

좁은 화면에서도 콘텐츠가 잘리지 않도록 세로 흐름 유지

0

강제 스크롤 차단 없이 콘텐츠 높이에 맞춰 자연스럽게 이동

Palette

모래색 배경과 녹색, 오렌지 포인트를 조합해 차분하지만 심심하지 않게 맞췄습니다.

Layout

데스크톱에서는 카드가 퍼지고, 모바일에서는 한 줄 흐름으로 접혀 짧은 화면에서도 안전합니다.

Focus

링크와 버튼에는 `:focus-visible` 윤곽선을 넣어 키보드 탐색도 깨지지 않게 유지했습니다.

작업 기준

QA 피드백의 핵심은 뷰포트보다 긴 페이지에서 `overflow: hidden`이 실제 콘텐츠를 가려버린다는 점이었습니다. 이 페이지는 스크롤을 막지 않고, 높이가 늘어나도 그대로 읽을 수 있게 두었습니다.
